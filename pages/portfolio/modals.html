Auction platform

WeAuction is auction platform project consisting of 3 parts.

Management - management of auctions and objects as well as crerking part of the auction floor.

Actual auction web page - allow tracking auctions and related details.

Auction widget - Real time updated custom element that can be installed on any web page. Auction widget is used to place a bid or track current object state and price.
